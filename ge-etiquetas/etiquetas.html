Aprende HTML y CSS con Código Vivo

Bienvenido al entorno de desarrollo definitivo. Aquí tienes un desglose del comportamiento de las etiquetas estructurales, propiedades de diseño y cómo se entrelazan para dar vida a interfaces web profesionales.

Estructura General de HTML5

HTML (HyperText Markup Language) estructura el esqueleto de cualquier sitio. Estas son las etiquetas fundamentales organizadas por funcionalidad:

Etiqueta Categoría Función Principal
<html> Raíz Envuelve todo el documento web. Indica el inicio y fin del documento.
<head> Metadatos Contiene información invisible para el usuario (SEO, links a CSS, codificación UTF-8).
<body> Estructura Contiene toda la parte visual, textos, imágenes y secciones legibles de la web.
<header>, <nav>, <footer> Semántica Definen el encabezado, menú de navegación y pie de página de forma legible para buscadores.

Formatos y Contenedores

<h1> a <h6> Títulos. Va desde el más importante (h1) al menos importante (h6).
<p> Párrafos. Define bloques de texto estándar con espaciados automáticos.
<div> Contenedor genérico en bloque. Usado exclusivamente para aplicar estilos CSS.
<span> Contenedor genérico en línea. Útil para dar estilo a palabras específicas dentro de un texto.

Propiedades Esenciales de CSS

CSS (Cascading Style Sheets) maneja toda la capa visual. Se basa en reglas compuestas por un selector y propiedades de diseño.

Propiedad CSS Uso Ejemplo / Explicación
display: flex; Diseño Flexbox Alinea elementos hijos en filas o columnas de forma dinámica y responsiva.
grid-template-columns Diseño CSS Grid Crea rejillas o cuadrículas bidimensionales perfectas con filas y columnas exactas.
margin vs padding Modelo de Caja Margin añade espacio externo a la caja; Padding añade espacio interno.

Laboratorio: Mini Proyectos Prácticos

A continuación se muestran combinaciones reales de estas etiquetas estructurales y de diseño mediante componentes UI aislados.

Proyecto 1: Tarjeta de Perfil Profesional

Uso de: Box-Shadow, Bordes redondeados, Flexbox y Márgenes negativos.


<!-- HTML Estructural -->
<div class="p1-card">
    <div class="p1-banner"></div>
    <div class="p1-avatar"></div>
    <div class="p1-info">
        <div class="p1-nombre">Alex Dev</div>
        <div class="p1-puesto">Full Stack Engineer</div>
        <button class="p1-btn">Seguir</button>
    </div>
</div>

/* CSS Aplicado */
.p1-card { background: #ffffff; border-radius: 15px; width: 300px; text-align: center; overflow: hidden; }
.p1-banner { height: 100px; background: linear-gradient(135deg, #667eea, #764ba2); }
.p1-avatar { width: 90px; height: 90px; border-radius: 50%; margin: -45px auto 1rem auto; border: 4px solid white; }
.p1-btn { background: #4f46e5; color: white; padding: 0.6rem 1.5rem; border-radius: 20px; cursor: pointer; }

Proyecto 2: Formulario de Registro con Input Focus

Uso de: Selectores de pseudoclase (:focus), Inputs de texto y transiciones suaves.


<!-- HTML Estructural -->
<div class="p2-form-container">
    <div class="p2-group">
        <label class="p2-label">Correo Electrónico</label>
        <input type="email" class="p2-input" placeholder="tu@email.com">
    </div>
    <button class="p2-btn">Crear Cuenta</button>
</div>

/* CSS Aplicado */
.p2-form-container { width: 100%; max-width: 400px; background: #f8fafc; padding: 2rem; border-radius: 12px; }
.p2-input { width: 100%; padding: 0.75rem; border: 2px solid #cbd5e1; border-radius: 6px; transition: border 0.3s; }
.p2-input:focus { border-color: #06b6d4; } /* Cambio de color animado */
.p2-btn { width: 100%; background: #06b6d4; color: white; padding: 0.8rem; border-radius: 6px; font-weight: bold; }
