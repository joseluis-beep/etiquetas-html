Módulo de Etiquetas HTML

Etiqueta <div>

Elemento estructural base que define una división o sección en bloque dentro del cuerpo de la web.

<div style="background: #f43f5e; color: white; padding: 15px; border-radius: 6px; text-align: center;">\n    Este es un contenedor DIV con estilos de bloque en línea.\n</div>
Este es un contenedor DIV con estilos de bloque en línea.

Módulo de Propiedades CSS

Propiedad display: flex;

Habilita el modelo de caja flexible (Flexbox) para alinear componentes hijos de forma horizontal o vertical automáticamente.

<div style="display: flex; justify-content: space-between; background: #e2e8f0; padding: 10px;">\n    <span style="background: blue; color: white; padding: 5px;">A</span>\n    <span style="background: red; color: white; padding: 5px;">B</span>\n</div>
